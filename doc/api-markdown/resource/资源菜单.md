##### 请求头参数
> | 参数       | 必填 | 描述            |
> | ---------- | :--- |  --------------- |
> | Content-Type |true|请求体编码类型,使用:application/json;charset=UTF-8|

```
    token:"01fbec36c0b617cbaea00a89ccc08380"
}
```

##### 响应体参数
> | 参数       | 必选 | 类型 | 说明            |
> | ---------- | :--- | :--- | --------------- |
> | code |true|Integer|状态码|
> | data |true|list|返回数据|
> | data.uid |true|String|模型唯一标识|
> | data.name |true|String|模型名称|
> | message |true|String| |
